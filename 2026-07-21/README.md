# CookPilot design lab assets · 2026-07-21

This directory contains visual exploration artifacts referenced by the long-running GitHub Discussion `CookPilot 디자인 실험실`.

## Layout experiments

- `L001-Culinary-Editorial`: editorial home and vertical-step cooking layout
- `L002-Midnight-Hands-Free`: cinematic home and voice-orb cooking layout

## Skin experiments

All `Sxxx` screens share one interaction structure and differ only in visual treatment.

- `S001-Standard`: neutral accessible default
- `S002-Editorial-Skin`: parchment and brick-red editorial treatment
- `S003-Midnight-Skin`: graphite and mint cinematic treatment

These files are discussion artifacts, not production UI assets. The `design-lab-assets` branch is intentionally kept separate from the product branches.
